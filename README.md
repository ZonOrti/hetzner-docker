## Installation

Add this line to your application's Gemfile:

    gem 'hetzner-docker'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hetzner-docker

## Usage

```
export HETZNER_USER="" && export HETZNER_PASSWORD="" 

hetzner-docker bootstrap -i <server-ip> -d <server-domain>
```
